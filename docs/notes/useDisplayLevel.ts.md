## 表示のモード切替に関して

### モード

- 建物ラベル
- 入口出口オンリー
- 詳細表示

### 建物名表示: "building"

- 半透明オン
- 建物名ラベル表示
- 詳細ラベル非表示

### 入口出口の表示: "entrance"

- 半透明オフ
- 建物名ラベル非表示
- 詳細ラベル非表示
- 入口出口アイコン表示

### 詳細表示: "detail"

- 半透明オフ
- 建物名ラベル非表示
- 詳細ラベル表示
- 入口出口アイコン表示
