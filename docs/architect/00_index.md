# 00. 設計書インデックス

## 本書について

本書は FrontierAtlas 屋内測位システムの全設計書のインデックスである。各ドキュメントの要約・依存関係・推奨読込順を提供する。

## 図表の共通凡例

各ドキュメント内の図表で使用される記法：

| 記法 | 意味 |
|------|------|
| `→` | データの流れ / 処理の順序 |
| `⇒` | 非同期処理の流れ |
| `◄─►` | 双方向通信 |
| `┌──┐` 等 | コンポーネント境界 |
| `【】` | 通信方向ラベル |
